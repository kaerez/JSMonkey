Advanced Geolocation Manager
============================
[Return to home page](/README.md)

<br>

Prerequisite
--------------------

[Tampermonkey](https://www.tampermonkey.net/)

<br>

Installation
------------

Click [here](https://raw.githubusercontent.com/kaerez/JSMonkey/main/GeoManager/advanced-geo-manager.user.js) to install this script

<br>

Use
---

Browse to any webpage, then: right-click on an empty place in the page -> 📍 Advanced Geolocation Manager (or access it via the Tampermonkey extension menu). \
A management interface will appear where you can create spoofing profiles, fetch real coordinates, set target website rules, and run diagnostics.
