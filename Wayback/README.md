Wayback
=======
[Return to home page](/README.md)

<br>

Prerequisite
--------------------

[Tampermonkey](https://www.tampermonkey.net/)

<br>

Installation
------------

Click [here](https://raw.githubusercontent.com/kaerez/jsmonkey-pub/main/Wayback/wayback.user.js) to install this script

<br>

Archiving a URL
---------------

Browse to the URL you want to archive, then: right click on an empty place in the page -> Tampermonkey -> Wayback
